### Control

|     Key       | Action        |
| ------------- | ------------- |
| left arrow,a             | move left  |
| right arrow,d             | move right  |
| up arrow, w             | rotate  |
| down arrow, s        | move down  |
| SPACE, ENTER             | activate  |
| p           | pause, resume  |
| ESC           | quit  |
