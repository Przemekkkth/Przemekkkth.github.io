### Control

|     Key       | Action        |
| ------------- | ------------- |
| A             | decrease shot angle  |
| S             | increase shot angle  |
| Z             | jump  |
| SPACE         | make fire  |
| O             | make screenshot  |
| ESC           | quit app  |
