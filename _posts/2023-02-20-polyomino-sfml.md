### Control

|     Key       | Action        |
| ------------- | ------------- |
| left arrow             | move left  |
| right arrow             | move right  |
| up arrow, z             | rotate  |
| down arrow         | move down  |
| SPACE, ENTER             | activate  |
| ESC           | pause, resume  |
